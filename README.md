# Pandas_Project_1

- Pandas 라이브러리를 활용하여 산불 데이터 분석을 수행하였습니다.
- 화재 발생 원인, 화재 발생 시간 및 진화 시간 피해 규모, 지역, 날짜를 분석하였습니다.
- 저는 그 중에서 지역을 중심으로, 시도별 산불 발생 빈도, 시도별 대형산불위험예보목록에 대한 분석을 진행하였습니다.
